¿De qué va esto?

El Web Games Day reúne en un día las novedades en la plataforma Web que impulsan el desarrollo y la distribución moderna de videojuegos.

Se trata de un evento para todos los entusiastas de videojuegos, técnicos o no, que se interesen en la plataforma universal, libre y carente de fricción que es la Web.

Este Web Games Day nace como la culminación de una pequeña colaboración piloto entre el grado de programación de video juegos de la Facultad de Informática de la UCM y el equipo de DevRel de Mozilla. Nuestra intención es la de mostrar cómo la Web apoya y promueve tecnologías clave en el desarrollo y la distribución moderna de video juegos.

A esta colaboración se unen las comunidades MadridJS, HTML Spain y Video Game Army con el fin de difundir el evento, que se presta de interés no sólo para entusiastas de los video juegos sino para todos aquellos que disfruten o trabajen con tecnologías Web.

Las charlas

La agenda está a punto de confirmarse y podemos ir adelantando que incluirá charlas sobre realidad virtual con WebVR, rendimiento casi nativo por medio de Web Assembly y sonido de la mano de Web Audio, entre otros.

Building VR experiences

WebVR is an API to create Virtual Reality experiences right in the browser. We also have some tools that let us create there experiences in a much more easy and productive way, like A-Frame, an open-source VR framework. In this talk we will learn how to use A-Frame, as well as its possibilities.

The Web Audio API

Audio with web technologies... it doesn't always or ever have to be annoying background music that starts playing when you open a new tab.

The Web Audio API is extremely powerful, we can create sounds, load them, analyse them, modify them in an insane amount of different ways.

This is an in-depth talk into the different methods of the API and not just that but how best we would use them in our development work flow. What techniques really come in useful when loading and manipulating sounds? This talk looks to outline just that and you should go away feeling more confident knowing which methods to use when.
